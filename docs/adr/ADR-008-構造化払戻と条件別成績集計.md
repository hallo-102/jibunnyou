# ADR-008: 構造化払戻と条件別成績集計

- 状態: 採用
- 決定日: 2026-07-11

## 決定

- 結果状態を provisional、confirmed、cancelledに分け、暫定結果は精算しない。
- 払戻は券種、正規化組合せ、100円基準払戻、normal/refundを構造化保存する。
- 候補の各組合せを券種込みで照合し、複数的中と返還を合算する。
- 旧 `payout_amount` は合計払戻として後方互換読取し、新規結果は `payouts_json` を正とする。
- 精算へ券種、予想source、的中数、的中組合せ、払戻内訳、結果状態を保存する。
- 成績は期間、source、券種、競馬場、course、class、Python model、AI modelで絞込・集計する。
- 的中率、回収率、損益、最大連敗、最大ドローダウンをBackendで計算する。
