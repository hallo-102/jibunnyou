## 質問

`keibayosou_best_import_roi_runner.py` で予想して出力するExcelの `買い目_レース別1行` シートについて、既存のU列の軸馬入力ロジックとV〜Y列の `1頭軸_相手`、`1頭軸_金額`、`保険BOX_馬番`、`保険BOX_金額` を除外し、U列を `軸馬番` として新条件で選んだ軸馬番を入れたい。条件は rank <= 3、単勝オッズ <= 12.0、extra_penalty <= 1.5、期待値 >= 0.80、危険馬判定なし。優先順位は危険馬判定なし、単勝オッズ8倍以下優先、extra_penalty0.5以下優先、期待値は0.8以上でよい。

## 回答要約

- `keibayosou_pipeline.py` の `買い目_レース別1行` 生成から旧 `1頭軸_*` と `保険BOX_*` の計算・列出力を削除し、U列用の `軸馬番` プレースホルダーだけを出すようにした。
- `keibayosou_best_import_roi_runner.py` に、推定馬券内率・期待値・危険馬シート作成後に `買い目_レース別1行` の `軸馬番` を更新する後処理を追加した。
- 軸馬番はランキング1〜3位から条件を満たす馬だけを対象にし、危険馬スコア3以上相当の馬を除外したうえで、単勝オッズ8倍以下、extra_penalty0.5以下、予想順位、単勝オッズ、extra_penalty、score、期待値の順で選ぶようにした。
- 条件に合う馬がいないレースは `-` を入力するようにした。
